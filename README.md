# CodeQuiz

Test your JavaScript knowledge using this fun quiz!

Press Start, and then answer a series of questions to the best of your ability. You will have 75 seconds to complete the quiz. Be careful - you will lose time if you answer a question incorrectly!

Once you are complete, you can save your initials to the High Score board. To view saved High Scores, click "View High Scores" at any time. 

Happy quizing!
